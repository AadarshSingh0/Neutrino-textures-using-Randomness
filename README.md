# Random Graph-Mass-Models
All the codes required to reproduce the results int he arxiv paper are uploaded here. The Hopping.nb and Site.nb files containt he code for diagonal and off-diagonal randomness in the mass matrix. The corresponding
.m package files for reusable functions is also provided. The module_compiler.nb file is used to retrive function from these .m files and to produce data for various counts. The other two .nb files are for
comparison between clockwork and random models and for different graph lattices (local/non-local/Petersen). Plots.nb file is to generate the plots as are seen in the paper.
